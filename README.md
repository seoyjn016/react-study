# CSS 기초

- html 은 화면에 보여줄 데이터(글자) 입니다.
- css는 화면에 보여줄 데이터를 보기 좋게 꾸며주는 역할을 합니다.

## 1. css 작성법 (4가지)

### 1.1. 인라인 방식 : html 직접 적용

- <태그 style="이름 : 값">
- 가독성이 떨어져요.

```html
<body style="background: hotpink"></body>
```

### 1.2. <style> 태그 활용하기

- 가독성은 좋아요.
- css 코드 재활용은 어렵습니다.
- 선택의 대상 { css 적용 }

```html
<style>
  body {
    background: hotpink;
  }
</style>
```
